\doxysubsection{Detailed Description}
\doxylink{class_command}{Command} to count rows in a table that satisfy a specific column-\/value condition. 

This command enables users to query a table and determine how many rows have a particular value in a designated column. The result, which is the total count of matching rows, is displayed to the console.

Executes the count command, determining the number of matching rows. 

Executes the \textquotesingle{}count\textquotesingle{} command, tallying rows based on a column condition.

This method counts the occurrences of a specified {\ttfamily search\+Value} within a {\ttfamily search\+Column\+Index} of a given {\ttfamily table\+Name}. It retrieves the table from the loaded catalog, iterates through the relevant column, and increments a counter for each cell that matches the criteria. The final count is then printed to the console.

This function processes the command-\/line arguments to determine the target table, the column to search within, and the value to match. It performs validation checks to ensure a catalog is loaded, parameters are correct, and the table exists.

The method then retrieves the specified table and column. It iterates through the cells of the {\ttfamily search\+Column\+Index}. For each cell, it checks if the value matches the {\ttfamily search\+Value}. A counter is incremented for every match found. Finally, the total count of matching rows is printed to the console in a user-\/friendly format.
