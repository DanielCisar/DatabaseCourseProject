\doxysubsection{Detailed Description}
Manages a collection of database tables and their associated catalog file path. 

The {\ttfamily \doxylink{class_catalog}{Catalog}} class represents the in-\/memory database. It holds a collection of {\ttfamily \doxylink{class_table}{Table}} objects, providing functionalities to add, remove, and retrieve tables by name. It also tracks the file path on disk where the catalog\textquotesingle{}s metadata is or will be stored. The {\ttfamily \doxylink{class_catalog}{Catalog}} relies on the {\ttfamily \doxylink{class_table}{Table}} class\textquotesingle{}s copy semantics for proper deep copying of table data.
