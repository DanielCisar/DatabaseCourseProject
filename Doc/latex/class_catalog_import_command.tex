\doxysubsection{Detailed Description}
\doxylink{class_command}{Command} to import a new table from a file into the currently loaded catalog. 

This command allows users to load external table data, typically from a CSV file, and integrate it into the application\textquotesingle{}s in-\/memory database catalog. The imported table becomes available for further operations within the application.

Executes the \textquotesingle{}import\textquotesingle{} command, loading a table from a file. 

Executes the \textquotesingle{}import\textquotesingle{} command, loading a table from a specified file into the catalog.

This method reads a table from the specified {\ttfamily filepath} and adds it to the {\ttfamily context.\+loaded\+Catalog}. The imported table\textquotesingle{}s name is derived from the file\textquotesingle{}s content (or is implicitly part of the \doxylink{class_table}{Table} object from {\ttfamily read\+Table\+From\+File}). A success message is printed to the console upon successful import.

This function processes the command-\/line arguments to obtain the file path of the table to be imported. It performs validation checks to ensure a catalog is loaded and the correct number of parameters (2) is provided.

The method then uses {\ttfamily \doxylink{class_input_file_reader_a876332636cc935a29dda58b03edf631b}{Input\+File\+Reader\+::read\+Table\+From\+File}} to parse the table data from the specified {\ttfamily filepath}. The resulting {\ttfamily \doxylink{class_table}{Table}} object is then added to the {\ttfamily context.\+loaded\+Catalog}. This operation makes the new table part of the active in-\/memory database. A confirmation message, including the imported table\textquotesingle{}s name, is printed to the console upon success. Any errors during the process are caught and reported.
