\doxysubsection{Detailed Description}
Concrete implementation of {\ttfamily \doxylink{class_table_column}{Table\+Column}} for storing integer data. 

This class manages a column of integer values, including support for NULL values. It provides methods for accessing, modifying, adding, and deleting integer cells, performing necessary string-\/to-\/integer conversions. It strictly adheres to the polymorphic interface defined by the {\ttfamily \doxylink{class_table_column}{Table\+Column}} base class. NULL values are typically represented internally by a default value (e.\+g., 0) in {\ttfamily content} and a corresponding {\ttfamily true} in the {\ttfamily is\+Null} boolean vector.
