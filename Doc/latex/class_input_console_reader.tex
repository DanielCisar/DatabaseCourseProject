\doxysubsection{Detailed Description}
Handles reading input from the console. 

This class provides utility methods to read lines of text from standard input ({\ttfamily std\+::cin}), either as a single string or parsed into a vector of strings based on whitespace or a specified delimiter. It\textquotesingle{}s designed to abstract the console input mechanism.
