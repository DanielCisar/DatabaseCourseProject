\doxysubsection{Detailed Description}
Concrete implementation of {\ttfamily \doxylink{class_table_column}{Table\+Column}} for storing string data. 

This class manages a column of string values, including support for NULL values. It provides methods for accessing, modifying, adding, and deleting string cells, and adheres to the polymorphic interface defined by the {\ttfamily \doxylink{class_table_column}{Table\+Column}} base class. NULL values are represented internally by an empty string in {\ttfamily content} and a corresponding {\ttfamily true} in the {\ttfamily is\+Null} boolean vector.
