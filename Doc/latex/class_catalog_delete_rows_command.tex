\doxysubsection{Detailed Description}
\doxylink{class_command}{Command} to delete rows from a table that match a specific column-\/value condition. 

This command enables users to remove rows from a designated table. It searches for rows where a specified column\textquotesingle{}s value matches a given criterion and deletes all such identified rows.

Executes the delete rows command, removing matching records from a table. 

Executes the \textquotesingle{}delete\textquotesingle{} command, removing rows from a table based on a condition.

This method searches for rows in the specified {\ttfamily table\+Name} where the value in {\ttfamily search\+Column\+Index} matches {\ttfamily search\+Value}. All rows that satisfy this condition are then deleted from the table. A count of the deleted rows is displayed to the console.

This function processes the command-\/line arguments to identify the target table, the column to search within, and the value to match. It performs validation checks to ensure a catalog is loaded, parameters are correct, and the table exists.

The method then retrieves the specified table and column. It iterates through the rows of the table in reverse order (from last to first) to safely delete rows without affecting subsequent iteration indices. For each row, it checks if the value in the {\ttfamily search\+Column\+Index} matches the {\ttfamily search\+Value}. If a match is found, the entire row is deleted from the table, and a counter is incremented. Finally, the total count of deleted rows is printed to the console.
