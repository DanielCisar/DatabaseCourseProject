\doxysubsection{Detailed Description}
A static utility class for parsing command strings into tokens. 

{\ttfamily \doxylink{class_command_parser}{Command\+Parser}} provides methods to break down a single input string (representing a command and its arguments) into a collection of individual tokens (words or phrases). It supports parsing by a custom delimiter and a specialized parsing for raw commands that handles quoted strings.

Parses a string into a vector of tokens using a specified delimiter. 

This method iterates through the input {\ttfamily line} and splits it into substrings wherever the {\ttfamily delimiter} character is found. Each substring becomes a token in the returned vector. Consecutive delimiters will result in empty tokens.

Parses a raw command string, specifically handling spaces and quoted arguments. 

This method is designed to parse commands from user input where arguments might contain spaces and be enclosed in double quotes.
\begin{DoxyItemize}
\item Spaces outside of double quotes act as delimiters.
\item Text inside double quotes is treated as a single token, including spaces.
\item Double quotes themselves are not included in the resulting tokens.
\item Multiple consecutive spaces outside quotes will generate empty tokens.
\end{DoxyItemize}
