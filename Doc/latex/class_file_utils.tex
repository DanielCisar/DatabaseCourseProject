\doxysubsection{Detailed Description}
A static utility class for common file system operations. 

{\ttfamily \doxylink{class_file_utils}{File\+Utils}} provides convenient methods to check for file existence and extract directory paths from full file paths. It\textquotesingle{}s designed to offer cross-\/platform basic file system utilities.

Extracts the directory path from a full file path. 

This method finds the last occurrence of a path separator ({\ttfamily /} or {\ttfamily \textbackslash{}}) in the given {\ttfamily filepath} and returns the substring up to that point, effectively giving you the directory where the file resides. If no path separator is found, it assumes the file is in the current directory and returns "{}."{}.
