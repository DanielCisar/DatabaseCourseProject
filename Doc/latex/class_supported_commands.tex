\doxysubsection{Detailed Description}
A static utility class that provides lists of recognized commands. 

This class serves as a central repository for defining and retrieving the names of commands supported by the application, categorized by their functional area (e.\+g., file operations, catalog operations). It\textquotesingle{}s useful for displaying help menus and for validating user input.
