\doxysubsection{Detailed Description}
\doxylink{class_command}{Command} to rename an existing table within the loaded catalog. 

This command allows users to change the name of a specific table that is currently loaded in the application\textquotesingle{}s database catalog. The operation updates the table\textquotesingle{}s identifier in memory.

Executes the \textquotesingle{}rename\textquotesingle{} command, changing the name of a table in the catalog.

This method renames a table from {\ttfamily old\+Name} to {\ttfamily new\+Name} within the {\ttfamily context.\+loaded\+Catalog}. It retrieves the table by its current name and updates its internal name property. A confirmation message is printed to the console upon successful renaming.

\begin{DoxyNote}{Note}
This command directly changes the table\textquotesingle{}s name property. If the {\ttfamily \doxylink{class_catalog}{Catalog}} class relies on unique table names as keys in an internal map, an external mechanism in {\ttfamily \doxylink{class_catalog}{Catalog}} might be required to update its internal mapping for full consistency (e.\+g., a {\ttfamily Catalog\+::rename\+Table} method). As implemented, it only changes the {\ttfamily \doxylink{class_table}{Table}} object\textquotesingle{}s internal name.
\end{DoxyNote}
This function processes the command-\/line arguments to obtain the current (old) name of the table and its desired new name. It performs validation checks to ensure a catalog is loaded and the correct number of parameters (3) is provided.

The method then retrieves the specified table from the {\ttfamily context.\+loaded\+Catalog} using its {\ttfamily old\+Name}. If the table is found, its {\ttfamily set\+Name} method is called to update its internal name property. A confirmation message is printed to the console upon successful renaming. Any errors (e.\+g., table not found) are caught and reported.
