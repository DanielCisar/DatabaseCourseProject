\doxysubsection{Detailed Description}
Represents a single table within the database catalog, holding a collection of columns. 

The {\ttfamily \doxylink{class_table}{Table}} class manages the structure and data of a database table. It internally holds a collection of polymorphic {\ttfamily \doxylink{class_table_column}{Table\+Column}} pointers, effectively owning the column objects and being responsible for their memory management (deep copying on copy/assignment, deletion on destruction). It provides functionalities for accessing column and row data, modifying the table structure, and iterating over its columns.
