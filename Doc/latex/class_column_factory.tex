\doxysubsection{Detailed Description}
A static factory class responsible for creating concrete {\ttfamily \doxylink{class_table_column}{Table\+Column}} instances. 

The {\ttfamily \doxylink{class_column_factory}{Column\+Factory}} provides a centralized and type-\/safe way to instantiate different types of columns (String, Integer, Double) without exposing their specific constructors directly to client code. This promotes decoupling and simplifies the creation process for {\ttfamily \doxylink{class_table_column}{Table\+Column}} objects.
