\doxysubsection{Detailed Description}
A static factory class responsible for creating {\ttfamily \doxylink{class_table_column}{Table\+Column}} instances. 

The {\ttfamily \doxylink{class_column_factory}{Column\+Factory}} provides a centralized safe way to instantiate different types of columns (String, Integer, Double) without exposing their specific constructors directly to client code.
