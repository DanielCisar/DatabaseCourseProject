\doxysubsection{Detailed Description}
\doxylink{class_command}{Command} to insert a new row of data into a specified table. 

This command allows users to add a new record to an existing table in the loaded database catalog. It requires the table name and a list of values, which must correspond to the number and types of columns in the table.

Executes the \textquotesingle{}insert\textquotesingle{} command, adding a new row to a table. 

Executes the \textquotesingle{}insert\textquotesingle{} command, adding a new row to a specified table.

This method adds a new row of data to the table identified by {\ttfamily params\mbox{[}1\mbox{]}}. The values for the new row are taken from {\ttfamily params\mbox{[}2\mbox{]}} onwards. The number of provided values must exactly match the number of columns in the target table. Each value is added to its corresponding column, potentially involving type conversion based on the column\textquotesingle{}s type.

\item {\ttfamily params\mbox{[}2\mbox{]}}...{\ttfamily params\mbox{[}N\mbox{]}}\+: The values for each column in the new row, in order corresponding to the table\textquotesingle{}s columns.

This function processes the command-\/line arguments to obtain the target table name and a list of values for the new row. It performs validation checks\+:
\begin{DoxyEnumerate}
\item Ensures a catalog is loaded.
\item Verifies that at least the table name is provided ({\ttfamily params.\+size() \texorpdfstring{$>$}{>}= 2}).
\item Retrieves the specified table from the {\ttfamily context.\+loaded\+Catalog}.
\item Ensures that the number of provided values exactly matches the number of columns in the table.
\end{DoxyEnumerate}

The method then iterates through the table\textquotesingle{}s columns and, for each column, calls its {\ttfamily add\+Cell} method with the corresponding value from the input parameters. This effectively adds a new row to the table. A success message is printed to the console upon successful insertion. Any errors (e.\+g., table not found, wrong number of values, type conversion failures) are caught and reported to the console.
