\doxysubsection{Detailed Description}
A static utility class for common input/output operations, particularly for paginated display. 

{\ttfamily \doxylink{class_i_o_utils}{IOUtils}} encapsulates reusable functionality for interacting with the console, specifically providing a method to display a large set of data in a user-\/friendly, paginated format, allowing the user to navigate through pages.

Prints a vector of strings to the console in a paginated format. 

This method displays the provided {\ttfamily rows} data page by page. It pauses after each page, prompting the user to enter commands like "{}next"{}, "{}prev"{}, or "{}exit"{} to navigate through the pages. The page size is fixed at 10 rows.
