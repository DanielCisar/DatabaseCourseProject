Detailed Description

Concrete implementation of TableColumn for storing string data.

This class manages a column of string values, including support for NULL values. It provides methods for accessing, modifying, adding, and deleting string cells, and adheres to the polymorphic interface defined by the TableColumn base class. NULL values are represented internally by an empty string in content and a corresponding true in the isNull boolean vector.
