Detailed Description

Manages a collection of database tables and their associated catalog file path.

The Catalog class represents the in-memory database. It holds a collection of Table objects, providing functionalities to add, remove, and retrieve tables by name. It also tracks the file path on disk where the catalog's metadata is or will be stored. The Catalog relies on the Table class's copy semantics for proper deep copying of table data.

void Catalog::addTable ( Table table)

Adds a new table to the catalog.

Before adding, this method checks if a table with the same name already exists in the catalog to enforce uniqueness. If the name is unique, the table is added as a copy into the internal collection.

Parameters
tableThe Table object to add. It is passed by value to ensure a copy.
Exceptions
std::runtime_errorIf a table with the same name already exists in the catalog.

Before adding, this method iterates through the existing tables to ensure that no other table with the same name already exists. If a duplicate name is found, an exception is thrown. Otherwise, the provided Table object (which is a copy of the original passed by value) is moved into the tableCollection.
