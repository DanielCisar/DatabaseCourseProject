Detailed Description

Handles the "exit" file command which terminates the program.

This command is responsible for displaying a goodbye message and signaling the main loop to stop (typically handled elsewhere after execution).

Outputs a message to notify the user that the program is exiting. This function does not terminate the program itself — it is expected that the main Engine loop will check for this command and handle termination.
