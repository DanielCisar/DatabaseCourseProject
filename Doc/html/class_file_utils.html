Detailed Description

A static utility class for common file system operations.

FileUtils provides convenient methods to check for file existence and extract directory paths from full file paths. It's designed to offer cross-platform basic file system utilities.

bool FileUtils::fileExists ( const std::string & path)
static

Checks if a file or directory exists at the specified path.

This method attempts to open the file at the given path to determine its existence and accessibility. It returns true if the file exists and is in a "good" state (e.g., can be opened), and false otherwise.

Parameters
pathThe file system path to check.
Returns
true if the file exists and is accessible; false otherwise.

This implementation uses std::ifstream to attempt opening the file. The file.good() method then checks if the stream is in a good state, which implies the file exists and can be accessed.

Parameters
pathThe file system path to check.
Returns
true if the file exists and is accessible; false otherwise.

std::string FileUtils::getDirectoryPath ( const std::string & filepath)
static

Extracts the directory path from a full file path.

This method finds the last occurrence of a path separator (/ or \) in the given filepath and returns the substring up to that point, effectively giving you the directory where the file resides. If no path separator is found, it assumes the file is in the current directory and returns ".".

Parameters
filepathThe full path to a file (e.g., "C:/data/documents/file.txt" or "data/file.txt").
Returns
A std::string representing the directory path. Returns "." if no directory is specified.

This function finds the last occurrence of either a forward slash (/) or a backslash (\) to determine the directory separator. If no separator is found, it assumes the file is in the current directory and returns ".".

Parameters
filepathThe full path to a file.
Returns
A std::string representing the directory path.
